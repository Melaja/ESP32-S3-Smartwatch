1. WHAT IS THIS PROJECT?
================================================================================

This project is a fully functional smartwatch application built on the
Waveshare ESP32-S3 Touch AMOLED 2.06" module. It combines a graphical
user interface (LVGL) with direct display control (Arduino_GFX) for a
rich multimedia experience on a compact AMOLED screen.

The smartwatch provides:
  - Digital clock with NTP time synchronization
  - Step counter with accelerometer
  - System settings (brightness, battery, WiFi)
  - File browser with SD card support
  - Multimedia viewer (images, audio, text)
  - Power saving with automatic dimming and sleep


2. HARDWARE PLATFORM
================================================================================

  Component               Specification
  ----------------------- --------------------------------------------------
  Microcontroller         ESP32-S3 (QFN56), Dual-core LX7 @ 240MHz
  Flash Memory            16 MB (QIO mode)
  PSRAM                   8 MB (OPI mode)
  Display                 CO5300 AMOLED, 410x502 pixels, 16-bit RGB565
  Display Interface       QSPI (Quad SPI)
  Touch Controller        FT3168, capacitive, I2C @ 0x38
  Power Management        AXP2101 PMIC, Li-Ion/Li-Po charger
  Audio Codec             ES8311 (DAC/ADC), I2C @ 0x18
  Audio ADC               ES7210 (microphone input)
  Power Amplifier         External, controlled via GPIO46 (PA_CTRL)
  IMU / Accelerometer     QMI8658 (6-axis: accelerometer + gyroscope)
  SD Card                 MicroSD via SPI (HSPI bus)
  Wireless                WiFi 802.11 b/g/n + Bluetooth 5.0 LE
  USB                     USB-Serial/JTAG (CDC mode)


3. SCREENS AND NAVIGATION
================================================================================

The application has 4 main screens plus a viewer screen:

  Screen 0: CLOCK         Time, date, day name (Dutch)
  Screen 1: STEPS         Step counter with pause/reset
  Screen 2: SETTINGS      Brightness, battery, WiFi status
  Screen 3: FILES         SD card file browser
  Viewer:   VIEWER        Overlay for images/audio/text

Navigation:
  - Swipe left:           Next screen (0 -> 1 -> 2 -> 3 -> 0)
  - Swipe right:          Previous screen (0 -> 3 -> 2 -> 1 -> 0)
  - Tap on file:          Open in viewer
  - 3 sec long press:     Return from viewer


4. POWER SAVING
================================================================================

Three phases of power saving:

  Phase       Trigger               Brightness    Updates
  ----------- --------------------- ------------- ---------------------
  ACTIVE      After any touch       200/255       Everything active
  DIMMED      15 sec inactivity     50/255        Everything active
  SLEEP       30 sec inactivity     0/255         Updates paused

Any touch interaction resets the timer and wakes the display immediately.


5. SUPPORTED FILE TYPES
================================================================================

  Type          Extensions             Viewer           Color in browser
